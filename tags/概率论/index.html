采样随机变换(Sampling Random Variables)：逆变换采样(The Inversion Method)

动机

实际使用随机变量时，我们希望能够在给定的分布中采样随机变量，这样就可以用更快收敛的方式用蒙特卡洛积分来估计积分。

步骤

  1. 计算累积分布函数CDF:$P(x)=\int_0^xp(x’)dx’$。
  2. 求CDF的反函数(inverse) $P^{-1}(x)$。
  3. 使用均匀分布的随机变量采样得到$\xi$。
  4. $X_i=P^{-1}(\xi)$。
